Radius Tutorials - Navigation

Undoubtedly the biggest greatest enhancement in the modernized Radius application is in how you interact with it. A complete UI / UX overhaul delivers the ability to quickly jump to the pages you need, creating a seamless workflow (unlike the classic Radius system where you had to constantly navigate to different windows and tabs).

The following information details the major changes in how you find your way around the Radius application, and gives high-level instructions and tips on how to get the most out of the new design. 

The Landing Page 

The first page presented to you when you open the modernized Radius application is the landing (Sales Order) page. This page has been designed so that you can quickly and easily find the records you need and navigate to the relevant area of the application in as few interactions as possible. 

Like the other Radius pages, the landing page is able to be customized and each customization can be saved as a view. 

The search bar at the top of the screen is used to quickly filter which sales order records display in the sales order browser. Alternatively, you can click Advanced Search  next to the search bar to access the Advanced Search Filters window, where you can leverage the Advanced Search engine to build powerful queries to filter records by (which can then be saved for quick access).

For detailed instructions on using the advanced search, see Advanced Search.

 

 

Key Information

In the classic Radius system, you would have to right click a record in a browser, then hover over Key Information before selecting the shortcut you needed.

This process has been simplified so that you can not just hover over the More  button next to a record in the browser to access the key information menu. From here you can navigate to pages and open them in the context of the respective sales order, or choose to edit or duplicate the sales order directly. 

Hover over the More  button  for quick access 

For even quicker access click the hyperlinked SO number

 

 

Sales Order Details, Shipping, Extras and More

Many of the windows you had to laboriously navigate through in the classic Radius application have been encapsulate within their relative location. For example, in addition to the shipping functionality receiving a streamlining, all the relevant fields and functionality are now directly accessible from the Sales Order (Add / Edit) page.

After successfully saving your sales order header, the following buttons are added to the bottom of the page which allow you enter the rest of your sales order information, such as order lines and shipping details:

  1. Order Lines
  2. Shipping
  3. Version Comments
  4. Extra
  5. Invoice Details
