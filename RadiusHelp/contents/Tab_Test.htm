Adding Tabbed Content

The following info is a quick tutorial on how to add tbs to your topic. It's a simple case of adding the div tags where you need them, I've set up the CSS and JS already. Just make sure you import the modernized.CSS stylesheet and the TabContent.js script.

I'll tidy up the CSS style as we go for the tabs, so they resize nicely for the screen / when there are lots of tabs etc.

Scripts don't run inside RoboHelp so you'll need to preview the topic in a table to test it. (ctrl + w shortcut, then click the eye at the top of the screen and select a table to launch in).

Adding Tab Buttons

To add a new tab, add a new button between the <div class=tablink-navigation>, for example:

<div class="tablink-navigation">

    <div style="text-align:center;">

        <button class="defualtopen tablink tablink-left" onclick="openTab(&#39;Tab1&#39;, this, &#39;#c3c3c3&#39;)">Tab 1</button>             

         <button class="tablink tablink-middle" onclick="openTab(&#39;Tab2&#39;, this, &#39;#c3c3c3&#39;)">Tab 2</button> 

        <button class="tablink tablink-right" onclick="openTab(&#39;Tab3&#39;, this, &#39;#c3c3c3&#39;)">Tab 3</button> 

    </div>

</div>

For the first tab, use class="tablink tablink-left", for midle tabs use  "tablink tablink-left" and for tabs on the end right, use  "tablink tablink-right". This is because the corners of the end tabs have a curve in the CSS whereas the middle ones do not.

 

Adding Tab Content

You need to then add the following code under the <div class="tabcontent">, in the space where there new tab is, for example:

<div class="tabcontent" id="Tab1">

    Place your content here

 </div>

<div class="tabcontent" id="Tab2">

   Place your content here

</div>

 

<div class="tabcontent" id="Tab3">

  Place your content here

</div>

 

By default the content for the tabs is hidden so it won't appear in authoring mode. This is because when the page loads the  openDefaultTabs(); method is called, which opens all tabs with a class name of defaultopen.



To show your content, simply change the following code in Modernized.CSS from



.tabcontent {

  color: white;

  display: none;

}



to



.tabcontent {

  color: white;

  display: block;

}



Just remember to change it back once you're done!
