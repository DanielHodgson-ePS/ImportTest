Radius Tutorials - Navigation

The biggest change when moving to the modernized Radius application is in how you interact with Radius itself. A complete UI / UX overhaul delivers the ability to quickly jump to the pages you need, enabling a seamless workflow, unlike the classic Radius system where you had to tediously navigate to different windows and tabs.

The following information details the major differences in how you find your way around the Radius application, and gives high-level instructions and tips on how to get the most out of the new design. 

The Landing Page

The first page presented to you when you open the modernized Radius application is the landing (Sales Order) page. This page has been designed in such a way that you can quickly and easily find the records you need, and navigate to the relevant area of the application in as few interactions as possible. 

Like the other Radius pages, the landing page is highly customizable, and each customization can be stored as a view for future use. 

Note: For detailed instructions on crating views, see How to Create Views and Layouts.
Note: For detailed instructions on managing views, see Managing Views and Layouts.

 

Searching

The search bar at the top of the screen is used to quickly filter which sales order records display in the sales order table. Alternatively, you can click  Advanced Search  next to the search bar to  access the Advanced Search Filters window, where you can leverage the Advanced Search engine to build powerful queries to filter records by (which can then be saved for quick access).

Note: For detailed instructions on using the advanced search, see  About the Advanced Search.

Navigating Sales Orders

Whereas to open a sales order record in the classic Radius software you had to manually navigate to the Search tab of the Sales Orders window, perform a search, select the record to open and then navigate to the tab you needed, you can now just simply click the hyperlinked sales order number in the table of the landing page. 

Note: For a full breakdown of the Sales Order (Add / Edit) page fields, see Sales Orders (Add/Edit Mode).

Upon opening the Sales Orders (Add / Edit) page, the the Sales Order Details section is expanded to display your sales order header information. Additional information such as sales order analytics, user defined elements, interface order details and intercompany transaction details are accessed by using the buttons above the Sales Order Details fields. 

Sales Order Details, Shipping, Extras and More

Many of the windows you had to laboriously navigate through in the classic Radius application have been encapsulate within their relative location. For example, in addition to the shipping functionality receiving a streamlining, all the relevant fields and functionality are now directly accessible from the Sales Order (Add / Edit) page.

After successfully saving your sales order header, the following buttons are added to the bottom of the page which allow you enter the rest of your sales order information, such as order lines and shipping details:

  1. Order Lines
  2. Shipping
  3. Version Comments
  4. Extra
  5. Invoice Details
 

Key Information / Quick Navigation

In the classic Radius system, you could right-click a record in a table, then hover over Key Information before selecting the shortcut you needed.

This process has been simplified so that you can now just hover over the More  button next to a record in the table to access the key information menu. From here you can navigate to pages and open them in the context of the respective record, or choose to edit or duplicate the sales order directly. 

Hover over the More  button   for quick access
