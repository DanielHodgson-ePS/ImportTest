File and Link Attachments

Use the File and Link Attachments window to attach or link files to a sales order. This provides a way of including supplementary information.

To display this window, hover over More Items  under the Sales Order Details section of the Sales Orders page and click  Files Attachments.

Note:  By default, Radius allows.pdf, .xls, .xlsx, .doc, .docx, .png, .jpeg and .img file formats to be uploaded. The maximum number of files that can be uploaded is also limited to five, with a maximum file size of 10MB for each file. However, you can change these settings in the Admin Settings page or circumvent them by alternatively adding a weblink to the files. See Configuring File Attachment Settings for more details.

When your changes are complete, click Done to close this window.

Add Files

Use this section to attach files, or add weblinks and file folders to the sales order record. 

Note: If these buttons are available, you may have already exceeded the size limit for attachments. 

Options

Description

Files

Weblinks and File Folders

Select one of the following options:

  1. Files: Select to attach files.

  2. Weblinks and File Folders: Select to enter a weblink or path to a file folder. 

Adding Files

When the Files option is selected, the following options are available:

Button/Option

Description

Select

Click to browse for and select the files you want to attach to the sales order record. 

Note: Alternatively, drag and drop the files you want to attach into the Add Files section.

Upload

After selecting the files by clicking Select or dragging and dropping the files into the Add Files section, click this to upload and attach them to the sales order.  After the files are uploaded, they are removed from the Add Files section and added to the Existing Files & Links section. 

Note: Any text entered in the Notes field will be added to the upload. 

Cancel

Click to remove files from the Add Files section so they are not available to upload and attach to the sales order record. .

Adding Weblinks and File Folders

When the Weblinks and File Folders option is selected, the following options are available.

Field/Button

Description

Add Files (text box)

Click to browse for and select the files you want to attach to the sales order record. 

Note: Alternatively, drag and drop the files you want to attach into the Add Files section.

Add (button)

After entering the weblink or file folder path, click this button. The weblink or file folder path is added to the to the Existing Files & Links section.

Note: Any text entered in the Notes field will be added to the upload.  

Notes (Optional)

Field

Description

Notes

Enter any additional notes in regards to the files you are uploading or the weblink you are adding. Then click Upload to upload the files along with the note or click Add to add the weblink along with the note.
