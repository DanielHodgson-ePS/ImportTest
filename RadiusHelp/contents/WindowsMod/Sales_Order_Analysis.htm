Sales Order Analysis

Use the Sales Order Analysis window to enter sales order analysis codes against the analysis types linked to the sales order.  These codes are used for transaction analysis. 

To display this window, click Sales Order Analysis  on the Sales Order window. 

To select assign an analysis code to an analysis type, do one of the following:

  1. Click Edit All  to open up editing of all fields and then make multiple edits. 

  2. Click Edit  next to the section you want to edit then make the edit.  

When your edits are complete, click Save All .  To close the window, click Done. 

Field

 Description

Analysis Type

The analysis type linked to sales order in the Company > Analysis > Analysis Type/Codes window. 

Analysis Code

Select the code you want to associate with this analysis type. These codes are defined in the Analysis > Analysis Type/Codes window. 

Description

This field automatically populates with the analysis code
