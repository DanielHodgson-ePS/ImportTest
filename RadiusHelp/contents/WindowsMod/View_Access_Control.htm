View Access Control

Use the View Access Control window to specify which users have access to the views that you have created. View permissions can be assigned by view, user or user group. 

To display this window:

  1. On a page with a grid (forms), hover over the layouts drop-down  and select  View Access Control.  

  2. On a landing page, however over  More and select  View Access Control.  

Note: See Managing Views and Layouts for detailed instructions. 

By Views

To assign permission to a user or group by view, select By View and the following options display. 

Fields

Field

Description

View Context

Select the form/page you want to assign permissions for. 

Note: The landing page is named SalesOrderMaintenance.

View Name

Select the view you want to assign permissions for. 

Show By

Select one of the following to assign permissions at that level: 

  1. Users

  2. Groups

Groups

If you selected to show by groups then the following options display. 

Section

Description

Available Groups

This section lists the groups that are available to assign permissions to.

Select a group and click Assign  to move it to the Allowed Groups section.  This gives that group permission to see and use the selected view. 

Allowed Groups

This section lists the groups that have permission to use this view. 

To remove a group from this section, click click Assign  to move it to the Available Groups section.  

Users

If you selected to show by users then the following options display. 

Section

Description

Available Users

This section lists the users that are available to assign permissions to.

Select a group and click Assign  to move it to the Allowed Users section.  This gives that user permission to see and use the selected view. 

Allowed Users

This section lists the users that have permission to use this view. 

To remove a user from this section, click Assign  to move it to the Available Users section.  

By User

To assign permissions to multiple views at the same time by user, select By User and the following options display. 

Fields

Field

Description

Users

Select the user name of the user that you want to give access to mutliple views. 

Role

The role assigned to this user.

User Group

If the user is assigned to a user group then the group is listed here. 

Sections

If you selected to show by groups then the following options display. 

Section

Description

Available Views

This section lists the views that are available that you can select to give access to for the selected user. 

Select a view and click Assign  to move it to the Allowed Views section.  This gives the selected user permission to see and use the selected view. 

Allowed Views

This section lists the views that this user has permission to use.  

To remove a view from this section, click Assign  to move it to the Available Views section.  

By Groups

To assign permissions to multiple views at the same time by group, select By Groups and the following options display. 

Fields

Field

Description

User Groups

Select the name of the group that you want to give access to multiple views. 

Sections

If you selected to show by groups then the following options display. 

Section

Description

Available Views

This section lists the views that are available that you can select to give access to for the selected user group. 

Select a view and click Assign  to move it to the Allowed Views section.  This gives the selected user group permission to see and use the selected view. 

Allowed Views

This section lists the views that this user group has permission to use.  

To remove a view from this section, click Assign  to move it to the Available Views section.
