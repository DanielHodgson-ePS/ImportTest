Item Code

Use the Item Code window to select an existing item to add to a sales order line or create an item from a template to add to a sales order line. 

To display this window, click  in the Item Code field under Order Lines on the Sales Order page. 

When you have made your selections, click OK to close the window.

Filters

Select one of the following filters. 

Field

Description

From Item Code

Select to search for an existing item code. 

From Template

Select to create an item from an item template. 

From Item Code

When the Item Code option is selected, select the row containing the item you want to add to the order line and click OK. 

To search search for an item, enter your search criteria in the fields at the top and the grid automatically displays the search results. 

From Template

When the From Template option is selected, the following options display. 

Templates

Use the Templates section to select a template from which you want to create the item. When the template is selected, details of the template display in the Templates Field Values section to the right. 

To search for a template, enter your search criteria in the fields at the top and the grid automatically displays the search results. 

Field

Description

Code

The code for the template. 

Description

A description of the template. 

Template Field Values 

Use the Template Field Values section to see the selected values in the template. 

Entry Fields (Bottom Section) 

In the bottom section of the window, enter the following values for your new item. 

Field

Description

Item Code

Enter an appropriate item code for the new item. This must be unique at company level. 

Item Desc

Enter an appropriate description for the material. This description will be shown on inquiries, lookups and purchase orders. If you want to force this short description to be unique, select the appropriate option on the Items tab of the Company Level Options window.

Buttons

Button

Description

Cancel

Click to cancel selecting an item.

OK

Click to add the item to the sales order line.
