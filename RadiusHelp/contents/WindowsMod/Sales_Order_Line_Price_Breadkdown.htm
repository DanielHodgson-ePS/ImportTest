Sales Order Line Price Breakdown

Use the Sales Order Line Price Breakdown window to create a breakdown of the price of your sale sorder lines. To access this window, click the Sales Order Line Pricing button on the Order Lines tab of the Sales Orders (Add/Edit Mode) window.

Top Fields Section

Displays header information for the order line that the price breakdown is for.

Column 

Description

Slaes Order Number

Displays the sales order number.

Description

Displays the description the item.

Break Down Balance

Displays the total value of the price breakdown.

Customer Reference

Displays the customer code for the sales order.

Item Code

Displays the item code of the tiem on the order line.

Ordered Value

Displays the total value of the order line.

Price Breakdown Table Section

The price breakdown table displays price breakdown lines you have created for your sales order line. Click the Add New button to add a new line to the table, which can then be inline edited. When adding a price breakdown line you must specify values for the following fields:

  • Price Break Code: Specify the price break code that the price breakdown line is for. For example, Freight.
  • Breakdown Value: Enter the percentage of the slaes order line value that the price breakdown encompases. 

You can click the Edit All Lines  button to enable all the lines in the table for editing. 

Note: When updating a price breakdown line, only the Breakdown Value can be updated.
