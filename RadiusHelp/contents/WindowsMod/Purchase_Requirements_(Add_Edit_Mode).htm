Purchase Requirements (Add/Edit Mode)

Use the Purchase Requirements (Add/Edit) window to purchase items and materials by the requirements of a job or estimate.

Requirement Details

The Requirement Details section contains the main details of the purchase requirement record; holding the item, status, required quantity, job and other details.

Fields (Requirement Details)

Field

Description
Required For

Select what the requirement is for, choose from:

  • Issue
  • Shipment
  • Reorder
  • XLink Production Demand 
Status

Select the status of the production requirement. All new requirements have a status of Unplanned when created automatically which can be changed to Unauthorized if it needs to be held. The status will be automatically updated as you decide how to satisfy the requirement.
Item 

Specify the item the purchase requirement is for. 
Size

Enter the size of the Item.

Specify the date the goods must eb ready for.

Group

This field allows you to pass the requirement to another requirement group to be processed. For example: this may be an item that can either be made or purchased. The Item record will direct the initial requirement to either a Purchasing or Production Group. So if the Purchasing Department receive this first and they know that Production have capacity to produce the requirement, they can change the Group to a Production Requirement Group.
Job

Specify the job the requirement is for.
Step Number

Specify the step number the requirement is for.
