Extra Comments

Use the Extra Comments window to add comments to your sales order extras information.

To display this window, click Add from template in the Sales Orders (Add/Edit Mode) window.

Click the Add from template  button in this window to open the second Sales Order Comments window, where you can create and select template text for your comments.

First Extra Comments Window 

Fields / Buttons

Description

Comments (Field)

Enter comments relating to the sales order extra.

Add from template (button)

Click to open the second Extra Comments window, where you can create and select template text for your comments.

Cancel (button)

Click to disgard your comments and close the window. 

Add Comment (Button)

 Click to save your comments.

Second Extra Comments Window

Use the second Sales Order Comments window to select from a list of predefined text to add to your sales order comments.

Fields / Buttons

Description

Comments (Field)

Enter comments relating to the sales order extra.

Add from template (button)

Click to open the second Extra Comments window, where you can create and select template text for your comments.

Cancel (button)

Click to disgard your comments and close the window. 

Add Comment (Button)

 Click to save your comments.

Select a Template Text (Field)

Select which template to select predefined comments from, which are then displayed below this field. You can then click Add to Comments  next to predefined text you want to add to add it to the Comments field.
