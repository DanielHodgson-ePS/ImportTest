Admin Settings

Use the Admin Settings page to specify file attachment and output settings. 

To display this window, select

System Settings

File Attachment (File Attachment Settings)

Configure the settings for attaching files to records. 

Field

Description

File storage path on server

Specify the location where the files will be stored on the server.  Click Verify to confirm that the path is valid. 

Note: If you change this path then you will lose access to previously saved files. 

Max File Size

Move the slider to change the size of the files that users can attach to a sales order record. 

Number of files allowed to attach

Enter the the maximum number of files a user can attach to a record. 

File Formats

Enter the format of the files that users are allowed to attach to records. 

File Output (File Export Settings)

Configure the settings for exporting data to external files. 

Field

Description

Maximum number of records to export 

Enter the maximum number of records that can be export to an external Excel file.
