Intercompany Transaction Details

Click to open the Intercompany Transaction Details window where you can view details of the original sales order in the source company from which this sales order was based on.

This is only applicable if you are using intercompany transfers. This button is only enabled when the Intercompany transfers in use check box is selected on the General tab of the Company > Options > Company Level Options window.

When you click the Intercompany Transaction Details button, it displays the Intercompany Transaction Details window, which contains the following read-only fields:

Received from company: The linked company reference and company description for the company which sent the intercompany transfer request.

Received from plant: The linked plant code and plant description for the plant which sent the intercompany transfer request.

Purchase order: The purchase order number of the linked company which was sent to create this sales order.

Sales order: The sales order number of the linked company which started the intercompany transfer.

For more information, see About Intercompany Transfers.

  
View Quantities by Plant

Click to open the Quantities by Plant Window to allow you to see which plants the order has been allocated to.

  
User Defined Elements

Click this button to open the User Defined Elements Window where you can view user defined elements associated with the sales order. User Defined Elements are additional fields which you can define and add into the EFI Radius database. The purpose of the fields is to allow you to record additional information which can be viewed or added to customized reports. See Company > Jobs and Costing > User Defined Elements.

If this button is clicked in update mode changes can be made and saved by clicking OK as long as it passes any validation. If this button is clicked in view only mode, then the user defined elements window will only be displayed in view only mode as well.
