Sales Order

Use the Sales Orders > Sales Orders window to record customer’s purchase order details, agreed selling prices, extra charges and shipping details.

Sales Order details

Buttons

Button

 Description
 Sales Order Analysis Click to open the Sales Order Analysis window to maintain sales order analysis codes.  These codes are used for transaction analysis.
User Defined Elements Click to open the User Defined Elements where you can view user defined elements (custom fields) associated with the sales order. User Defined Elements are additional fields which you can define and add into the EFI Radius database. The purpose of the fields is to allow you to record additional information which can be viewed or added to customized reports. See Company > Jobs and Costing > User Defined Elements.
View Quantities by Plant Click to open the Quantities By Plant window to view which plants the order has been allocated to.
Interface Order Details Click to open the Interface Order Details window where you can maintain interface details when sales orders are created from a third party system.
Sales Order Notes Click to open the Sales Order Notes window
Files Attachment Click to open the File and Link Attachments
Intercompany Transaction Details Click to open the Intercompany Transaction Details window
