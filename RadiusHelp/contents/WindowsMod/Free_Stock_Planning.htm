Free Stock Planning

Use the Free Stock Planning window,a ccessed by clicking the Run Free Stock Planning button on the FSP tab of the Purchase Orders window, to process purchase requirements where it is not appropriate to reserve the requirements against specific inventory, purchase order lines or jobs.

You can select the requirements to be planned based on the following selections for item group, item, quarantine status, QA release status.

Field

Description

From Item Group

Specify the start range for the item group.

From Item Type

Specify the start range for the item type. A group has to be selected before specifying the item type.

From Item

Specify the start range for the item code.

To Item Group

Specify the end range for the item group.

To Item Type

Specify the end range for the item type. A group has to be selected before specifying the item type.

To Item

Specify the end range for the item code.

Cutoff date

Specify the planning run cut off date.

Plan Stock available

Enter the delay time in days when comparing the have ready date with the due date for purchase orders and the production receipts. The default setting is 1 day.

  • Days after expected delivery from supplier - Days to add to expected delivery date to determine planned available date.
  • Days after expected receipt from production - Days to add to expected receipt date to determine planned available date.

Generate Reorder Requirements (Check box)

Select to generate reorder requirements while running a stock planning.

Quarantined stock in planning calculation

Select whether to Exclude or Include quarantined stock in planning calculation.

Stock waiting QA Release in planning calculation

Select to exclude stock waiting QA release in planning calculation.

Generate Reorder Requirements (Check Box)

This is used when you have the Inventory re-order level field set to Company or Plant on the Inventory tab of the Company Level Options window. It allows you to process a range of items (by group, type or range of items) and generate reorder requirements based on item re-order level, minimum, maximum stock and re-order quantity.

Quarantined stock in reorder calculation

Select whether to Exclude or Include quarantined stock in reorder calculation.

Stock waiting QA Release in reorder calculation

Select whether to Exclude or Include stock waiting QA release in reorder calculation.

Buttons

Description

Plan

Click to run the Free Stock requirement planning details. A Question dialog box opens, click Yes.

Cancel

Click to cancel the free stock requirement planning. The Free Stock Planning window closes.
