Reservations

Use the Reservations window—accessed by clicking Reservations  in the Balances section of Order Lines tab of the Sales Orders (Add/Edit Mode) window—to add, change or delete reservations for the selected sales order line.

Top Fields Section

This section displays header information about the item on the reservation line selected form the below browsers.

Column 

Description

Order Line

Displays the sales order number.

Size

Displays the dimensions of the item.

Customer Reference

Displays the customer code for the sales order.

Item

Displays the item code.

Description

Displays the description the item.

Available for Reservation Section

Double-click on a line in the Available for Reservation section to move it to the Current Reservation section. Click the Done button to confitm your reservations.

Current Reservation Section

The Current Reservation table displays the current reservation details for the sales order line.

Job Line Balances Sections

The Job Line Balances sections display information for the line selected from the browser above it.
