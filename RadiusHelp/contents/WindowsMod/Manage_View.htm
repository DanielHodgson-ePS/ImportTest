Manage Views

Use the Manage Views window to define default views, delete views and share views.  This windows lists the views currently available. Use the options below to manage those views. The options available vary depending on where you view this window from. 

To display this window:

  1. On a page with a grid (forms), hover over the layouts drop-down  and select  Manage Views. 

  2. On a landing page, however over  More and select  Manage Views. 

Note: See Managing Views and Layouts for detailed instructions. 

Check Boxes

Button

Description

Select the check box to the left of the view to make it visible as a quick filter on the landing page.  This button is only applicable if you launched it from landing page. 

Buttons

Button

Description

 Share

Click to display the View Access Control window where you can assign user permissions at a user-level or group-level, enabling other users access to this view. 

 Mark to Delete

Click to mark a view for deletion. When you click Done or Apply, the view will be deleted. 

 Pin

Click to make this the default view when you open this page. 

 Move

Click this button and drag to the view you want to rearrange, and move it above or under another view. 

Note: This button is only visible if you launch this window from the landing page.
