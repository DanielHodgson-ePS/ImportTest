Add Ship Request from Order Lines

Click  Shipping Line under the Order Lines > Shipping Request Lines section of the Sales Orders page. 

By default the sales order lines from the currently open sales order are displayed. 

Select Order Lines

Search for and select the sales order lines to add to the shipping request. 

Search Section

The search criteria defaults to the currently open sales order.  If you want to search for sales order lines from another sales order, enter the criteria and click Search. 

Field

Description

Plant Code

Select the plant where the sales order was created. 

Outstanding

All

Select one of the following options: 

  1. Outstanding: Displays sales order lines that have not yet been shipped. It defaults to this selection. 

  2. All: Displays all sales order lines regardless of their status. 

Sales Order

Optionally specify the sales order number for the sales order lines you want to ship.  This defaults to the sales order that is currently open. 

Item Code

Optionally specify the item code that you want to ship. 

Results Section

This section displays the order lines based on the search criteria entered above. Select the check box to the left of the line to add it to the shipping request. 

To filter the results: 

  1. Click the Show All button to display the results of the search. This shows lines that are selected and unselected.

  2. Click the Show Selected button to only display order lines which have been selected. 

Click Next to move to the next page.  

Add Ship Request Lines

Complete the following information for the shipping request line. 

Column

Description

Required

Ship Quantity

 Enter the quantity to ship against this individual shipping request.

Yes

Customers Item Reference

Optionally specify the customer’s reference for the item. 

No

End Use

Specify the type of end product, such as food or pharmaceuticals.

This is used for determining the taxability in the OneSource integration when the item type or group type code cannot be used. 

No

Customer PO Line Reference

Optionally enter a line reference from the customer. This is sometimes required by some customers as a means of identifying the purchase order line in their computer system.

No

Customer PO Release Reference

Optionally enter a customer’s release reference for the shipping request.

No

Click Save to create the shipping request line and return to the Sales Orders page.
