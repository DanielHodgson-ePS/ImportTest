Add Ship Request from Order Lines

Click  Shipping Line under the Order Lines > Shipping Request Lines section of the Sales Orders page. 

By default the sales order lines from the currently open sales order are displayed. 

Select Order Lines

Search Section

The search criteria defaults to the currently open sales order.  If you want to search for sales order lines from another sales order, enter the criteria and click Search. 

Field

Description

Plant Code

Select the plant where the sales order was created. 

Outstanding

All

Select one of the following options: 

  1. Outstanding: Dipslays sales order lines that have not yet been shipped. It defaults to this selection. 

  2. All: Displays all ssles order lines regardless of their status. 

Sales Order

Optionally, specify the sales order number for the sales order lines you want to ship.  This defaults to the sales order that is currently open. 

Item Code

Optionaly, specify the item code that you want to ship. 

Results Section

This section displays the results order lines based on the search criteria entered above. Select the check box to the left of the line to add it to the shipping request. 

To filter the results: 

  1. Click the Show All button to display the results of the search. This shows lines that are selected and unselected.

  2. Click the Show Selected button to only display order lines which have been selected.
