Intercompany Transaction Details

Use the Intercompany Transaction Details to view details of the original sales order in the source company from which this sales order was based on. This is only applicable if you are using intercompany transfers. 

Intercompany transfer functionality allows you to place an order in one company, perform production and shipping in another (the destination company), and then perform billing in both companies. The destination company will bill the source company whilst the source company will bill the customer.

To display this window, click  Intercompany Transaction Details under the Sales Order Details section of the Sales Orders page. 

The fields in this window are read-only. Click Done to close this window. 

Exported Information

The destination company details are shown under the Exported Information section when the Intercompany Transaction Details window is displayed in the source company.

Note: The Exported Information section of the Intercompany Transaction Details window is only populated for the source company.

Field

Description

Sent to company

The linked company external reference and company description for the destination company which is performing the production.

Sent to plant

The linked plant code and plant description for the plant which is performing the production.

Sent via purchase order

The source company purchase order number that was used to send the order to the destination company.

Sales order

The sales order number in the destination company.  

Imported Information

The source company details are shown under the Imported Information section when the Intercompany Transaction Details window is displayed in the destination company.

Field

Description

Received from company

The linked company reference and company description for the source company which sent the intercompany transfer request.

Received from plant

The linked plant code and plant description for the plant which sent the intercompany transfer request.

Purchase order

The purchase order number of the linked source company which was sent to create this sales order.

Sales order

The sales order number of the linked source company which started the intercompany transfer.
