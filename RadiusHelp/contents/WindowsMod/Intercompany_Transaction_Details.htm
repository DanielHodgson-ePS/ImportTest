Intercompany Transaction Details

Use the Intercompany Transaction Details to view details of the original sales order in the source company from which this sales order was based on. This is only applicable if you are using intercompany transfers. 

To display this window, click  Intercompany Transaction Details under the Sales Order Details section of the Sales Orders page. 

The fields in this window are read-only. Click Done to close this window. 

Field

Description

Received from company

The linked company reference and company description for the company which sent the intercompany transfer request.

Received from plant

The linked plant code and plant description for the plant which sent the intercompany transfer request.

Purchase order

The purchase order number of the linked company which was sent to create this sales order.

Sales order

The sales order number of the linked company which started the intercompany transfer.
