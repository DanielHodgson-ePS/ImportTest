Sales Order Notes

Use the Sales Order Notes window to enter any notes you want to store against the sales order. 

To display this window, click  Sales Order Notes under the Sales Order Details section of the Sales Orders page. 

Note:  Click  in the top right corner of the window to close the window. 

Sections

Section

Description

Notes

Enter any new notes you want to store against the sales order record and then click Add to save the note.

Notes History

This section displays the existing notes that are stored against the sales order record.

Click delete  to remove an an existing comment.

Click edit  to display the existing note in the Notes section where you can modify the note and click Add. 

Buttons

Button

Description

Add

Click to add the new or amended note from the Notes field. 

Click to delete an existing note from the the Notes History section.

Click to edit an existing note in the Notes History section. When you click this button, the note displays in the Notes field where you can then edit the note and click Add.

Click to close the window.
