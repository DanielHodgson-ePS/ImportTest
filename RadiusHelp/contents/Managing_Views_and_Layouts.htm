Managing Views and Layouts

The modernized Radius application allows you to tailor build your Radius experience. You can now change the position of fields on the fly and decide which ones appear on your pages. Grids can be easily reordered with drag and drop functionality, and you can customize them to choose which columns they display.

You can easily share, edit and delete views at any time. Default views can be assigned so that a page will always open in that view, and you can even control who has access to a view by assigning permissions at user or group level.

While view management is largely the same for the landing page and forms, there are some slight differences. The following information gives instructions you on the many ways you can manage your Radius views from the landing page and from a form.

Saving a View

Click Save as view above the sales order grid to save the current page configuration as a view.  See How to Create Views and Layouts for detailed information on customizing and saving views.

Deleting a View

If you have a view that you no longer want to keep, you can easily delete it via the Manage Views window.

To delete a view, do the following:

  1. Hover over  More  next to your view icons and select Manage Views . The Manage Views window opens.
  2. Click Delete  next to the views you want to delete.
  3. Click the Apply button to delete the views.

Setting a Default View

If you have a view that you want to show whenever you load the landing page, you can set it as a default view in the  Manage Views window.

To set a default view, do the following:

  1. Hover over  More  next to your view icons and select Manage Views . The Manage Views window opens.
  2. Click Pin  next to the view you want to make default.
  3. Click the Apply button to set the view as default.

Rearranging a View

You can easily rearrange the order that your views appear on the landing page in the  Manage Views window.

To rearrange your views, do the following:

  1. Hover over  More  next to your view icons and select Manage Views . The Manage Views window opens.
  2. Click and drag  Move   next to the view you want to rearrange, and move it above or under another view.
  3. Repeat step 2 for each view you want to rearrange.
  4. Click the Apply button to accept your changes.

Saving a View

Click Save as View   above a grid or click Save Layout View   above a layout to save the current page configuration as a view.   See How to Create Views and Layouts for detailed information on customizing and saving views.

Deleting a View

If you have a view that you no longer want to keep, you can easily delete it via the Manage Views window.

To delete a view, do the following:

  1. Hover the layouts drop-down  and select Manage Views . TheManage Views window opens.
  1. Click Delete  next to the views you want to delete.
  2. Click the Apply button to delete the views.

Setting a Default View

If you have a view that you want to show whenever you load the page the form is on, you can set it as a default view in the  Manage Views window.

To set a default view, do the following:

  1. Hover the layouts drop-down  and select Manage Views . The Manage Views window opens.
  2. Click Pin  next to the view you want to make default.
  3. Click the Apply button to set the view as default.

Assigning View Permissions

After creating your Radius views, you can easily determine the users that have access to them via the View Access Control window. This window is the same regardless of whether you navigate to it via the landing page or a form.

View permissions can be assigned at user-level or group-level.

To assign permissions to your views, do the following

  1. Do one of the following to access the  View Access Control window, depending on whether you are on the landing page or a form page:
  1. If you are on the landing page, hover over  More  next to your view icons and select View Access Control . The View Access Control window opens.
  2. If you are on a form page, hover the layouts drop-down  and select View Access Control . The View Access Control window opens.
  1. Do one of the following to choose which level to assign permissions at:
  1. Select the Users option under Show By to assign permissions at user-level.
  2. Select the Group option under Show By to assign permissions at group-level.
  1. Select the form you want to assign permissions for from the View Context field. 

Note: The landing page is named SalesOrderMaintenance.

  1. Select the view you want to assign permissions for from the View Name field. 
  2. Select the check box next to the users / group you want to assign permissions for from the Available Users/Groups grid on the left-hand side.
  3. Click Assign , located in between the Available Users/Groups and Allowed Users/Groups grids. The users / groups are moved to Allowed Users/Groups grid on the right-hand side. 
  4. Click Save  to save your changes,
  5. Click the Done to close the window.
