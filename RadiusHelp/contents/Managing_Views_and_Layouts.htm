Radius Tutorials - Managing Views and Layouts

The modernized Radius application allows you to tailor build your Radius experience. You can now change the position of fields on the fly and decide which ones appear on your pages. Browsers can be easily reordered with drag and drop functionality, and you can customize them to choose which columns they display.

You can easily share, edit and delete views at any time. Default views can be assigned so that a page will always open in that view, and you can even control who has access to a view by assignign permissions at user or group level.

The following information gives instructions you on the many ways you can manage your Radius views.

Deleting a View

If you have a view that you no longer want to keep, you can easily delete it via the Manage Views window.

To delete a view, do the fol

  1. Click Customize Column  above the browser. The Customize Column List window opens.
  2. Select the check box next to the columns that you want to display in the browser.
  3. Optionally, select the Select All check box is you want to display every available column in the browser. 
  4. Click the Okay button to save your browser layout. 

The Customize Column List window can be be used to quickly customize your browser, using the same techniques as described in Step 2. The order of the fields in this window determine the order of your columns, and you can rearrange them by clicking and dragging Move Line . You can also quickly pin columns here by clicking Pin Column  .
