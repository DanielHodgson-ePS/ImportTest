Radius Tutorials - Advanced Search

The Radius Advanced Search utilizes the Elastic Search engine, allowing you to build and perform search queries that return results incredibly fast, even when searching large databases. You can access this window by clicking Advanced Search  next to the search field on the toolbar.

 

About Query Building

Search queries are built against a Field, for which you select a Type, Operator, and enter or select a value. Multiple criteria can be added for a query, allowing you to create complex searches. Each criteria you define is autimatically saved, so it can be easily used again, and can be edited or deleted form within the Advanced Search window.

 

Operators

Operators tell the Advanced Search to perform a specific mathematical, relational or logical operation and produce final result. Simply put, they are options you use to choose the type of search you want to perform.  The following operators are available to you:

 Range: The value must be between the specified limits.
