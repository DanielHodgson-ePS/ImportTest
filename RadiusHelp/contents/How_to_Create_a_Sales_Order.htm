Radius Tutorials - How to Create a Sales Order

EFI Radius is designed to handle a variety of sales order processing methods from a simple one order to one job to one shipment scenario, to more complex orders and shipping instructions.

A sales order consists of order header details - with the customer and overall details - and order lines - with the item and quantity. Each sales order may have one or many order lines.

Each ordered item is regarded as a requirement that your company has to satisfy. This may be done by creating a production job to make the item, purchasing the Item or allocating existing item inventory to meet the order requirement.

Step 1: Create From New or Create From Estimate

When creating a new sales order, you can choose to create it from scratch or from an estimate. Creating a sales order from new means that all the fields are blank as no prior information is available. Creating a sales order from an estimate instead uses those details on the estimate as a template, and automatically populates some of the fields in the Sales Order window. 

To create a sales order, do the following:

  1. On the Radius landing page (Sales Order) hover over the Create Sales Order button and either:
    1. Click the Create From Estimate button then specify an estimate to use as a template in the Estimate Code window.
    2. Click the Create New button.
  2. The Sales Order (Add / Edit) page opens. Continue to Step 2.

Step 2: Complete Sales Order Header Details

Next, you must enter the header details for your sales order. The header contains information such as the customer, the due date for the sales order, the status of the order and the payment method etc. 

To complete the sales order header details, do the following:

You can populate as much information as you require, however the following instructed fields are mandatory and you are prevented from saving until you populate them. For a detailed description of each field, see Sales Orders (Add/Edit Mode).

  1. Do the following in the Sales Oder Details section of the Sales Order (Add / Edit page):
    1. Specify the customer the order is for in the Customer field.
    2. Specify the date the sales order must be completed for in the Required Date field.
    3. Specify the date the sales order must be available to ship by in the Have Ready Date field. 
    4. Specify The credit terms used for the customer’s invoices and payments in the Terms Code field. 
    5. Complete any additional fields you require.
  2. Click the Save All button to save your header details. 

After successfully saving your sales order header, buttons are added to the bottom of the page which allow you enter the rest of your sales order information, such as order lines and shipping details.

  1. Optionally, if you want to attach a file to your sales order, continue to step 2a.
  2. Continue to step 3.

Step 2a: Attach Files to A Sales Order (Optional)

If you have any files you wish to attach to a sales order, you can do so via the File and Link Attachment window accessed via the Sales Order Details section of the Sales Order (Add / Edit page). 

By default, Radius allows.pdf, .xls, .xlsx, .doc, .docx, .png, .jpeg and .img file formats to be uploaded. The maximum number of files that can be uploaded is also limited to five, with a maximum file size of 10MB for each file. However, you can change these settings in the Admin Settings page or circumvent them by alternatively adding a weblink to the files. See Configuring File Attachment Settings for more details.

To attach files to a sales order:

  1. Hover over More Items  in the Sales Order Details section of the Sales Order (Add / Edit page) and select Files Attachment . The File and Link Attachment window opens. 
  2. Do one of the following to select files to attach files to the sales order, successfully attached files appear in the Existing files & Links section:
  1. Drag and drop the files you want to attach into the Add Files section and click the Upload button.
  2. Click the + Select button and specify the files you want to attach and click the Upload button. 
  3. Select the Weblinks and File Folders option, then paste the link to the external file in the Add Files section and click the Add button.
  1. Optionally, enter any notes about the files in the Notes field.
  2. Click the Done button to close the window.
  3. Continue to step 3.

 

Step 3: Add Sales Order Lines

Once you have completed your header information, you must then add order lines for the sales order. You can have as many order lines as you want but you must have a minimum of one. 

To add sales order lines, do the following:

  1. Click the Order Lines button at the bottom of the Sales Order (add / edit) page to display the order lines information.
  2. Click the + Order Line button to add a new line and display the order line entry fields. 
  3. Do the following to create a new order line:
    1. Specify the item the order line is for in the Item Code field. 
    2. Enter the amount to charge per unit of measure for the order line in the Price field.
    3. Specify the unit of measure for the item using the lookup adjacent the Price field. 
    4. Enter the quantity of the item in the Quantity field. 
    5. Specify the date the sales order must be completed for in the Required Date field. This field defaults to the Required Date on your header information.
    6. Specify the date the sales order must be available to ship by in the Have Ready Date field. This field defaults to the Have Ready Date on your header information.
    7. Click the Save All button to save the order line details. 
  4. Click the Back  button to return to the order lines browser, then repeat steps 1 to 3 for each order line you want to add. 

After adding order lines to your sales order you must then create a shipping request to sent the goods to the customer, as well as add any extra costs for things such as freight. 

For detailed instructions on crating a shipping request, see  How to Create a Shipping Request.

Step 4: Add Extras (Optional)

Optionally, you may want to add additional charges to your sales order for things such as freight or handling charges. The following information describes how to do so.  

You can populate as much information as you require, however the following instructed fields are mandatory and you are prevented from saving until you populate them. For a detailed description of each field, see Sales Orders (Add/Edit Mode).

To add extras to your sales order, do the following:

  1. On the Radius landing page (Sales Order) click the sales order number you want to add an extra for. The Sales Order (Add / Edit) page opens for the respective sales order. 
  2. Click the Extra button to display the extras browser.
  3.  Click the + Add Extra button to display the New Extra Charge section entry fields
  4. Do the following to add an extra charge:
    1. Select whether the extra is a charge or refund to the customer from the Invoice/Credit field. 
    2. Specify the customer the extra is for in the Customer field.
    3. Optionally, specify the sales order line the extra is for from the Sales Order - Line field.
    4. Specify a billing code in the Bill Code field. Billing codes are here used to analyze the charge for the shipping request.
    5. Enter the value to be invoiced or credited in the Price field and specify a unit of measure for using the adjacent lookup.
    6. The Value is usually automatically calculated when a Bill Code, Quantity and Price are entered. If this field is not automatically populated, enter the value of the extra charge.
    7. Specify the general ledger code to post the cost to in the GL Code field.
    8. Click the Save All button to create the extra. 
  5. To add further extras, click Back  then repeat steps 3 and 4.
