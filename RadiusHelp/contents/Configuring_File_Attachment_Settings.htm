Radius Tutorials - Configuring File Attachment Settings

EFI Radius allows you to attach files to your sales orders, as a way of including supplementary information. By default, Radius allows.pdf, .xls, .xlsx, .doc, .docx, .png, .jpeg and .img file formats to be uploaded. The maximum number of files that can be uploaded is also limited to five, with a maximum file size of 10MB for each file.

For detailed instructions on attaching files to a sales order, see step 2a in How to Create a Sales Order.

However, you can change these settings in the Admin Settings page or circumvent them by alternatively adding a weblink to the files. See Configuring File Attachment Settings for more details.

Change the File Upload Location

By default, any files uploaded to your sales orders are uploaded to ... . You can easily change this location to anywhere on your server. 

Changing your file upload location means that previously uploaded files become inaccessible and need to be uploaded again.

To change your file upload location: 

  1. Hover over the User  on the toolbar and select Admin Settings. The Admin Settings window opens. 
  2. Select the System Settings tab, then select File Attachment from the left-hand panel. The File Attachment Settings section opens.
  3. Enter the path where you want your sales order files to be uploaded to in the File storage path on server field. 
  4. Click the Verify button. If the path is valid, the button turns green and the message "Storage path is OK" displays. 
  5. Click the Save button to save your changes.

Change the Maximum File Size and Number Allowance

By default, files you upload are limited to 10MB in size, unless you have added them via a weblink. However this limit can be increased to a maximum of 100MB.

To change your file upload size limit:

  1. Hover over the User  on the toolbar and select Admin Settings. The Admin Settings window opens. 
  2. Select the System Settings tab, then select File Attachment from the left-hand panel. The File Attachment Settings section opens.
  3. Click and drag the Max File Size slider to your preferred limit.

  1. Enter the maximum number of files allowed to be attached to a sales order in the No. of files allowed to attach field.
  2. Click the Save button to save your changes.
 

Change the Allowed File Types

By default, radius allows.pdf, .xls, .xlsx, .doc, .docx, .png, .jpeg and .img file formats to be uploaded. However, you are able to configure Radius to allow or deny the attachment of any file type to sales orders.

To change the types of files you are able to attach to sales orders:

  1. Hover over the User  on the toolbar and select Admin Settings. The Admin Settings window opens. 
  2. Select the System Settings tab, then select File Attachment from the left-hand panel. The File Attachment Settings section opens.
  3. To remove a file type from the allowed list of extensions, click the cross next to the extension name in the File Formats field.
  4. To add a file type to the allowed list of extensions, type the extension into the File Formats browser and press Enter.

  1. Click the Save button to save your changes.
