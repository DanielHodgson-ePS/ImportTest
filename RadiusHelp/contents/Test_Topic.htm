Description

Sales Order Analysis

Click to open the Sales Order Analysis window to maintain sales order analysis codes.  These codes are used for transaction analysis.

User Defined Elements

Click to open the User Defined Elements where you can view user defined elements (custom fields) associated with the sales order. User Defined Elements are additional fields which you can define and add into the EFI Radius database. The purpose of the fields is to allow you to record additional information which can be viewed or added to customized reports. See Company > Jobs and Costing > User Defined Elements.

View Quantities by Plant

Click to open the Quantities By Plant window to view which plants the order has been allocated to.

Interface Order Details

Click to open the Interface Order Details window where you can maintain interface details when sales orders are created from a third party system.

Sales Order Notes

Click to open the Sales Order Notes window where you can enter any additional notes on the sales order. 

Files Attachment

Click to open the File and Link Attachments where you can attach or link files to a sales order. This provides a way of including supplementary information. 

By default, Radius allows.pdf, .xls, .xlsx, .doc, .docx, .png, .jpeg and .img file formats to be uploaded. The maximum number of files that can be uploaded is also limited to five, with a maximum file size of 10MB for each file. However, you can change these settings in the Admin Settings page or circumvent them by alternatively adding a weblink to the files. See Configuring File Attachment Settings for more details.

Intercompany Transaction Details

Click to open the Intercompany Transaction Details window to view details of the original sales order in the source company from which this sales order was based on.

This is only applicable if you are using intercompany transfers. This button is only enabled when the Intercompany transfers in use check box is selected on the General tab of the Company > Options > Company Level Options window.
