Radius Tutorials - How to Use the Advanced Search

The Radius Advanced Search utilizes the Elastic Search engine, allowing you to build and perform search queries that return results incredibly fast, even when searching large databases. You can access this window by clicking Advanced Search  next to the search field on the toolbar.

The following information gives detailed instructions on performing an Advanced Search, as well as how to delete and edit your saved queries.

Note: For a full breakdown and description of each element of the advanced search, see About the Advanced Search. Also see Advanced Search Filters for more information. 

Step 1: Create a Search Query

  1. Click Advanced Search  next to the search field on the toolbar. The Advanced Search Filters window opens.


  2. Do the following to build a search query:
    1. Select the field you want to query from Field field.
    2. Select the strictness of the query from the Type section. Choose from:
  1. Must: The results must equal the specified criteria.
  2. Must not: The results must not equal the specified criteria.
  3. Should: Prioritize the results of the search results that have been returned with optional criteria.
  1. Select a search operator for the value you are querying choose from the Operator column. Choose from:
  1. Range: The value must be between the specified limits.
  2. Match: The results have to contain this word.
  3. Exists: The estimate must have the specified field populated.
  4. Prefix: The word in the specified field must begin with this. If there are two words in the field then either word can begin with this.
  5. Fuzzy: The word in the specified field must be similar to this. This is useful for spelling mistakes.
  6. Wildcard: Use wildcards, meaning that a symbol can be used to replace or represent one or more characters. The most common wildcards are the asterisk (*), which represents one or more characters and question mark (?) that represents a single character. 
  1. If you selected one of the following options from the Operator section, enter or select a value to query for in the respective value field at the bottom of the window:
  1. Match
  2. Wildcard
  3. Prefix
  4. Fuzzy
  1. If you selected Range from the Operator field , do the following:
  1. Enter the minimum value to filter by in the From field.
  2. Enter the maximum value to filter by in the To field. 
  1. Click the  Add Criteria button and repeat Step 2 for each criteria you want to add to the query. Each criteria you add us automatically saved and added to the right-hand panel.
  2. Click the Search button to perform the search. The window closes and the Sales Orders page displays records filtered by your criteria.

Step 2: Save a Search Query

You can easily save the current page configuration as a view to capture your current search query for future use. for detailed instructions on creating and managing views see How to Create Views and Layouts and Managing Views and Layouts.

Step 3: Edit a Search Query

Each of your search criteria is displayed above the sales order records table. You can click Edit  next to the criteria to open it in the Advanced Search filter window or navigate to the Advanced Search Filter window and click click Edit  next to the criteria there. Refer to the instructions in Step 2 to adjust your criteria.

Step 4: Delete a Search Query

Each of your search criteria is displayed above the sales order records table. You can click Delete  next to the criteria to open it in the Advanced Search filter window or navigate to the Advanced Search Filter window and click click Delete  next to the criteria there. Refer to the instructions in Step 2 to adjust your criteria.
