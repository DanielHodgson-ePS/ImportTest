Radius Tutorials - How to Create a Shipping Request

Radius uses a header - lines relationship for its shipping requests, meaning that you can combine multiple sales orders onto a single shipping request and shipping note. This is useful in the situation where the customer calls-off different finished product items from several orders that are all to be shipped on the same day. Each request is for one address on one date.

Additionally, you can attach shipping lines to a header that were generated from different sales orders and plants.

This is due to Radius using a header - lines relationship for shipping requests. The header contains details of a shipment, such as the customer and the address, while the lines contain everything you want to send with that shipment.

Prerequisite: Create a Sales Order

The following instructions assume that you have already created a sales order that you want to create a shipment for. For a detailed description on how to create a sales order, see  How to Create a Sales Order.

Step 1: Create a Shipping Request Header

To create a Shipping Request, do the following:

Note: You can populate as much information as you require, however the following instructed fields are mandatory and you are prevented from saving until you populate them. For a detailed description of each field's purpose see  Sales Orders (Add/Edit Mode).

  1. On the Sales Orders (Landing Page) page click the sales order number you want to create a shipping request for. The Sales Order (Add / Edit) page opens for the respective sales order. 
  2. Click the Shipping button to display the Shipping Request Headers and Shipping Request Lines sections. 
  3. Click the  Shipping button to create a new shipping request header and expand the header detail entry fields. 
  4. Do the following to complete your shipping header details: 
    1. Specify the customer the shipment is for in the Customer field.
    2. Specify the plant the shipment is coming form in the Ship From Plant field.
    3. Specify when the shipment must leave the plant in the Ship Date & Time field.
  5. Click the Save All button to save the shipment header.
  6. Continue to Step 2.

The Shipping Request Headers and Shipping Request Lines sections of the Sales Orders (Add / Edit) page becomes available once you have created sales order lines. 

Note: Once a header or line is created, it can easily be duplicated by clicking Duplicate   next to it in the grid. 

Step 2: Add Shipping Request Lines

Next, you must add shipping lines to your header that contain the details of the items to be shipped. Shipping lines can be added from different sales orders, as well as different plants. You can add as many shipping lines as you need but must have a minimum of one.

To add shipping request lines, do the following:

Note: You can populate as much information as you require, however the following instructed fields are mandatory and you are prevented from saving until you populate them. For a detailed description of each field's purpose see Sales Orders (Add/Edit Mode) and Add Ship Request from Order Lines.

  1. Navigate to the Shipping Request Lines section of the Sales Order (Add / Edit page):
  2. Click the  Shipping Line button to add a new shipping request line. The Add Ship Request from Order Lines window opens. 
  3. Do the following in the Select Order Lines section to add shipping lines for your sales orders:
    1. Specify the sales order for the lines you want to add in the Sales Order field. By default the current sales order is populated. 
    2. Select the check box next to the sales order lines you want to create shipping lines for from the table below. 
    3. Click the Next button. The Add Ship Request Lines section opens. 
    4. Enter the quantity of item to add to the shipping line in the Ship Quantity field. 
    5. Click the Save button to save your shipping lines and return to the Shipping Request Lines section of the Sales Orders (Add / Edit) page. 
  4. Repeat steps 2 and 3 for until you have completed added your shipping lines. 

The Shipping Request Headers and Shipping Request Lines sections of the Sales Orders (Add / Edit) page becomes available once you have created sales order lines. 

Note: Once a header or line is created, it can easily be duplicated by clicking Duplicate   next to it in the table.
