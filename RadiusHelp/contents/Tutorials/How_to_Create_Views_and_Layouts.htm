Radius Tutorials - How to Create Views and Layouts

The modernized Radius application allows you to tailor build your Radius experience. You can now change the position of fields on the fly and decide which ones appear on your pages. grids can be easily reordered with drag and drop functionality, and you can customize them to choose which columns they display.

Once you have customized a page to how you want it, you can then save it as a view. Views can be shared with other users, and even set as default at an administrative level.

The following information gives instructions you on the many ways you can customize the layout in your Radius application.

Customizing a Grid

Radius uses grids to neatly display record information. You can in-line filter grids by entering a value in the search box at the top of the columns. Columns can be rearranged into an order that suits you best and you can even choose the columns that a grid displays.

Step 1: Choose Which Columns to Display

To choose the columns that display in a grid , do the following:

  1. Click  Customize Column above the grid.

    The Customize Column List window opens.
  2. Select the check box next to the columns that you want to display in the grid.
  3. Optionally, select the Select All check box is you want to display every available column in the grid. 
  4. Click the Okay button to save your grid layout. 

Note: The Customize Column List window can be be used to quickly customize your grid, using the same techniques as described in Step 2. The order of the fields in this window determine the order of your columns, and you can rearrange them by clicking and dragging Move Line . You can also quickly pin columns here by clicking Pin Column  .

Step 2: Rearrange Grid Columns

To rearrange the columns of a grid, do the following:

Note: You can switch between reverse sorting the column at any time by clicking Sort Column .

  1. To move a column into a specific location, click the column header and drag it next to the column where you want to place it. 

  1. To pin a column and move it to the far left of the grid (or the next available place if other columns have been pinned), click Pin this column . 

Step 3: Inline Filter Columns

To inline filter columns, do the following:

  1. If the field at the top of a column has a drop-down menu, simply select a value to filter by.
  2. If the field at the top of a column requires a value to be entered, enter a value to filter the column by. Results display immediately as you type. 
  3. Repeat steps 1 and 2 until the grid is filtered by the criteria you need. 

 

Step 4: Save the Grid Layout as a View

To save the changes you have made to the grid as a view, do the following: 

  1. Click  Save as View above the grid. The Save Layout View window opens. 
  2. Do one of the following depending on whether you want to create a new view or overwrite an existing one:
  • Select an existing view form the View Name field to overwrite it.
  • Enter a new name for the view in the View Name field to create a new view.
  1. Optionally, select the Make this as a default layout check box to create an icon for this view an icon that you can quickly select it from. 
  2. Click the Save button to save the grid layout as a view. 
 

Customizing a Form

Forms are the area of a page where you enter data, such as the Sales Order Details, New Shipping Request and New Extra Charge sections.

You can rearrange the position of fields, and choose which display by default by sorting them into custom tabs.

Step 1: Choose Which Columns to Display

To rearrange fields, do the following:

  1. Click  Rearrange Form Elements on the page you want to edit. The page opens in edit mode.
  2. Click and drag the form elements to rearrange them into the arrangement you want.

Note: Dragging a field into a tab section (for example, Additional Fields) will wide that field until that tab is expanded. This is useful for hiding the fields which you do not intend to use, allowing you to create a neater workspace.

  1. If you want to create custom tabs to arrange your form elements go to step 2, else go to step 3 for instructions on saving your form layout as a view. 

Step 2: Create Custom Tabs

Do the following to create custom tabs for your form elements:

  1. Click  Rearrange Form Elements on the page you want to edit. The page opens in edit mode.
  2. Click  New Tab  below your form elements to create a new tab. 
  3. Enter a name for your new tab and click  Accept .
  4. Go to step 3 to save your new form tab in a view. 

Note: Form elements are moved in and out of your custom tabs through the same method described in Step 1. Before you can delete a tab, you must first move all the elements out of it. To move form elements between tabs, you must first move it out of the tab into the main page, then select the tab you want to move it to and drag it there. 

Step 3: Save the Form Layout as a View

To save the changes you have made to your form as a view, do the following:

  1. Click  Save Layout View above the form fields. The Save Layout View window opens. 
  2. Do one of the following depending on whether you want to create a new view or overwrite an existing one:
  3. Select an existing view form the View Name field to overwrite it.
  4. Enter a new name for the view in the View Name field to create a new view.
  5. Optionally, select the Make this as a default layout check box to create an icon for this view an icon that you can quickly select it from. 
  6. Click the Save button to save the grid layout as a view.
