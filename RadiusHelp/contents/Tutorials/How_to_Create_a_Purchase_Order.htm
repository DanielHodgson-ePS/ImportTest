How to Create a Purchase Order

Purchase orders can be raised for a variety of purpose - from purchasing raw materials used in production to the purchasing of finished goods items themselves. Radius allows you to create purchase orders in a couple of different ways, giving you flexibility depending on your situation:

  • You can create a standard purchase order for one or more items.
  • You can create a purchase order to fulfill one or more purchase requirements.

Create a Standard Purchase Order

The following information details how to create a standard purchase order that does not hasve to fulfill any specific requirement. While purchase orders can be more complex (such as with the inclusion of price groups and version comments) the following information covers the creation of a basic purchase header order and adding order lines.

A purchase order is composed of header information and line information. The header contains details about the Purchase Order Number, Order Date and Suppleir Code, whereas the lines contain information about the items being purchased. 

Step 1: Create a Purchase Order Header

The following information details how to create a purchase order header to which you add your order lines.

  1. In Radius, hover over Purchase Orders  on the navigation pane and select Purchase Order. The Purchase Orders tab of the Purchase Orders page opens. 
  2. Click + Create Purchase Order. The Purchase Orders (Add/Edit Mode) page opens.
  3. Complete the fields in the Purchase Order Details and Additional Fields sections as required. 

Note: Mandatory fields are marked with a *.

  1. Optionally, complete the fields in the Comments and Addresses fields as required. 
  2. Click Save All to complete the purhcase order header information. The following tabs appear at the bottom of the page:
  • Order Lines
  • Price Group
  • Requirements and Reservations
  • Verison Comments

Step 2: Add Purchase Order Lines

After creating your purchase order header, you must then add lines for eahc item you wish to purchase.

  1. In Radius, hover over Purchase Orders  on the navigation pane and select Purchase Order. The Purchase Orders tab of the Purchase Orders page opens. 
  2. Click the Purchase Order Number in the table for the order you want to add lines to. The Purchase Orders (Add/Edit Mode) page opens.
  3. Select the Order Lines tab then do the following:
    1. Click + Order Line to add a line to the purchase order. The Purchase Order Lines (Add/Edit) section displays.
    2. Complete the fields in the Purchase Order Lines (Add/Edit) section as required. 
    3. Optionally, complete the fields in the Comments and Addresses fields as required. 
    4. Repeat steps a, b and c for each order line you want to add.
  4. Click Save All to save your order lines and complete the purchase order.

Convert Requirements to a Purchase Order

Purchase orders can also be raised by converting one or more purchase requirements. This can be accomplished through either the My Workspace (PR) tab or the Requirements tab of the Purchase Orders page. 

 

                

Convert Requirements to Purchase Order

The My Workspace (PR) tab acts as a hub to convinienty vire your requirements based on configurable filters. Your requiremnts can quickly be drag and dropped into the Add to PO Line column in this window, where you can then choose to either create a new purchase order for them or to add them to an existing purchase order. 

The following instructions detail how to convert a range of purchase requirements to a new purchase order.

  1. In Radius, hover over Purchase Orders  on the navigation pane and select Workspace - Requirements. The My Workspace (PR) tab of the Purchase Orders page opens. 
  2. Drag and drop each of the requirements you want to convert to a purchase order into the Add to PO Line column, then hover over Generate PO and select Create New PO. The PO Header tab of the Generate PO window opens.

 

  1. Do the following:

    Note: Mandatory fields are marked with a *.

    1. Complete the fields in the Purchase Order Details and Additional Fields sections as required. 
    2. Optionally, complete the fields in the Comments and Addresses fields as required. 
    3. Click Next. The Review Requirements tab opens. 
  2. Check the requirements on the purchase order header and deselect any as required.  
  3. Click Preview Line. The PO Line Preview tab opens. 
  4. For each of the purchase order lines on the left-hand side, do the following:

    Note: Mandatory fields are marked with a *.

    1. Select the purchase order line you want to enter details for. 
    2. Update the purchase order line fields as required.
  5.  Click Generate PO to create the purchase order.

 

Convert Requirements to Purchase Order

 The Requirements tab displays all of your requirements in a convienient searchable and customizable table. You can select a range of requirements from the table by clickign the check box next to the record, then choose to either create a new purchase order for them or to add them to an existing purchase order.  

The following instructions detail how to convert a range of purchase requirements to a new purchase order.

  1. In Radius, hover over Purchase Orders  on the navigation pane and select Requirements. The Requirements tab of the Purchase Orders page opens. 
  2. Select the check box next to each requirement you want to convert to a purchase order then click Create New PO. The PO Header tab of the Generate PO window opens.

 

  1. Do the following:

    Note: Mandatory fields are marked with a *.

    1. Complete the fields in the Purchase Order Details and Additional Fields sections as required. 
    2. Optionally, complete the fields in the Comments and Addresses fields as required. 
    3. Click Next. The Review Requirements tab opens. 
  2. Check the requirements on the purchase order header and deselect any as required.  
  3. Click Preview Line. The PO Line Preview tab opens. 
  4. For each of the purchase order lines on the left-hand side, do the following:

    Note: Mandatory fields are marked with a *.

    1. Select the purchase order line you want to enter details for. 
    2. Update the purchase order line fields as required.
  5.  Click Generate PO to create the purchase order.
