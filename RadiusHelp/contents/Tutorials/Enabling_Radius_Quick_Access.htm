Enabling Radius Quick Access

In the classic Radius system, you could right-click a record in a table, then hover over Key Information before selecting the shortcut you needed.

This process has been simplified so that you can now just hover over the More  button next to a record in the table to access the key information menu.

From the more menu, you can select the window you want to acess, which will then be opened in the classic Radius applciation (launching the application if it is not already open), using the context of the record currently selected in the respective table.

In order to use this quick access feature, you must ensure that the RadiusLauncher.exe file has been ran. The following instructions describe this process.

This action only needs to be completed once per Radius install.

How to Enable Radius Quick Access

 The following instructions detail how to run the RadiusLauncher.exe file to enable Ra dius quick access.

  1. Navigate to the Distrub folder of your Radius installation directory, for example C:\Radius\RadiusLive\Distrib.
  2. Open the RadiusLauncher folder.
  3. Right-click the RadiusLauncher.exe file and select Run as Administrator.
  4. A short automated configuration process runs. Once completed, radius quick access is enabled.
